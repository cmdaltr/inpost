# InPost MCP Configuration

This directory contains example MCP (Model Context Protocol) configuration for using Claude Desktop with Notion.

## Files

- `claude_desktop_config.json` — Example config snippet for Claude Desktop

## Setup

See [../docs/mcp-setup.md](../docs/mcp-setup.md) for full instructions.

## Quick Start

1. Copy the relevant section from `claude_desktop_config.json` into your Claude Desktop config
2. Replace `YOUR_NOTION_TOKEN` with your actual Notion integration token
3. Restart Claude Desktop
4. Ask Claude to interact with your Notion database
